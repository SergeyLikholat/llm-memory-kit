# Синхронизация между устройствами

Память и заметки — это текстовые файлы. Их удобно синхронизировать между
сервером, компьютером и телефоном. Рекомендуемый инструмент — **Syncthing**
(бесплатный, P2P, без облака). Для iPhone — отдельный путь (см. ниже).

## Syncthing — сервер ↔ ПК ↔ Android

| Устройство | Как |
|---|---|
| **Сервер (Linux)** | `apt install syncthing` или бинарь; запустить как сервис |
| **Windows** | [SyncTrayzor](https://github.com/canton7/SyncTrayzor) (Syncthing + трей) |
| **macOS** | `brew install syncthing` или [Syncthing-macOS](https://github.com/syncthing/syncthing-macos) — **работает нативно** |
| **Android** | приложение Syncthing из F-Droid/Play — **работает** |

Настройка: на каждом устройстве добавьте остальные по Device ID, расшарьте общую
папку (волт заметок и/или папку памяти). Синхронизация идёт напрямую, без облака.

## iPhone / iPad — не Syncthing

⚠️ **Syncthing на iOS официально не работает** — iOS не даёт приложениям фоновую
синхронизацию и доступ к файлам так, как нужно Syncthing. Сторонний **Möbius Sync**
существует, но фоновый синк ненадёжен. **Mutagen** — тоже не про iOS (это синк
серверов/десктопов через SSH).

**Рабочий путь для iOS — через git:**
- Папка памяти/заметок под git (kit держит Obsidian-волт под git автоматически).
- На iPhone — приложение **[Working Copy](https://workingcopy.app/)** (git-клиент):
  делает `pull`/`push` markdown-слоя. Идеально для текста.
- Альтернатива для заметок — платный **Obsidian Sync** (официальный, работает на iOS).

## Что синхронизировать

- **Лёгкий текст** (заметки .md, память) — синкайте везде, включая телефон.
- **Тяжёлый бинарь** (PDF, книги, большие Excel) — можно исключить из синка на телефон
  через `.stignore` (Syncthing) по типам файлов, чтобы не тянуть гигабайты.
